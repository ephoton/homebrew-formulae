Add this tap and trust it:
```bash
brew tap ephoton/homebrew-formulae
brew trust ephoton/homebrew-formulae
```

Then you can install any version of the [UndertaleModCLI](https://github.com/UnderminersTeam/UndertaleModTool/tree/master/UndertaleModCli):

```bash
brew install undertale-mod-cli@0.9.0.0
```
